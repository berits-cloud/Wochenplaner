Daten sichern

Deine Daten werden automatisch in diesem Browser gespeichert. Für ein Backup oder den Wechsel auf ein anderes Gerät kannst du eine Sicherungs-Datei speichern.

Du erhältst eine Datei „Mein-Wochenplaner.html" mit allen deinen Daten. Zum Wiederherstellen einfach diese Datei öffnen (Doppelklick) — kein Importieren nötig. So kannst du sie auch auf ein anderes Gerät kopieren.
